prepareQuery :: String -> [Text] Source #

Splits a query into segments, where each segment must appear in order in any matching names.

>>> prepareQuery "foo bar baz"
["foo","bar","baz"]

Split camel-case style words into segments. >>> prepareQuery "fMap" ["f",Map]

Collapse multiple spaces >>> prepareQuery "foo barBaz boom" ["foo","bar",Baz,"boom"]

Split namespaces into segments with a required dot in between. >>> prepareQuery "List.map" [List,".","map"]

Shouldn't get multiple splits for capitalized letters >>> prepareQuery List.Map [List,".",Map]
