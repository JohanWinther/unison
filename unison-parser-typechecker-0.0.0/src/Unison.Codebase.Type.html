data GitPushBehavior
  = -- | Don't set root, just sync entities.
    GitPushBehaviorGist
  | -- | After syncing entities, do a fast-forward check, then set the root.
    GitPushBehaviorFf
  | -- | After syncing entities, just set the root (force-pushy).
    GitPushBehaviorForce
