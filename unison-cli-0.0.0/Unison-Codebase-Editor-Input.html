type LooseCodeOrProject = These Path' (ProjectAndBranch (Maybe ProjectName) ProjectBranchName) Source #

A lot of commands can take either a loose code path or a project branch in the same argument slot. Usually, those have distinct syntaxes, but sometimes it's ambiguous, in which case we'd parse a These. The command itself can decide what to do with the ambiguity.
